# สรุปข้อมูลสินค้าทดสอบ

## ภาพรวม
สร้างสินค้าทดสอบทั้งหมด **50 รายการ** แบ่งเป็น 5 หมวดหมู่ หมวดหมู่ละ 10 รายการ

## รายละเอียดตามหมวดหมู่

### 1. เสื้อผ้า (14 รายการ)
- ราคา: ฿300 - ฿1,000
- สต็อก: 15-50 ชิ้น
- รายการ:
  - เสื้อยืดคอกลม สีขาว (฿300)
  - เสื้อเชิ้ตแขนยาว สีฟ้า (฿500)
  - เสื้อโปโล สีแดง (฿400)
  - เสื้อฮู้ดดี้ สีเทา (฿600)
  - เสื้อกันหนาว สีดำ (฿800)
  - เสื้อยืดคอวี สีเขียว (฿300)
  - เสื้อเชิ้ตลายสก๊อต (฿500)
  - เสื้อคาร์ดิแกน สีครีม (฿700)
  - เสื้อแจ็คเก็ต สีน้ำเงิน (฿1,000)
  - เสื้อสเวตเตอร์ สีชมพู (฿600)

### 2. รองเท้า (10 รายการ)
- ราคา: ฿200 - ฿2,500
- สต็อก: 15-60 ชิ้น
- รายการ:
  - รองเท้าผ้าใบ สีดำ (฿1,000)
  - รองเท้าหนัง สีน้ำตาล (฿2,000)
  - รองเท้าแตะ สีดำ (฿200)
  - รองเท้าบูท สีน้ำตาล (฿2,500)
  - รองเท้าวิ่ง สีขาว (฿1,500)
  - รองเท้าส้นสูง สีแดง (฿1,200)
  - รองเท้าแตะหนัง สีน้ำตาล (฿500)
  - รองเท้าผ้าใบ สีขาว (฿900)
  - รองเท้าโลฟเฟอร์ สีดำ (฿1,600)
  - รองเท้าแตะสุขภาพ (฿600)

### 3. กระเป๋า (10 รายการ)
- ราคา: ฿400 - ฿3,000
- สต็อก: 10-40 ชิ้น
- รายการ:
  - กระเป๋าสะพายข้าง (฿1,500)
  - กระเป๋าเป้ สีดำ (฿1,000)
  - กระเป๋าถือ สีแดง (฿2,000)
  - กระเป๋าคาดเอว สีเทา (฿500)
  - กระเป๋าเดินทาง 20 นิ้ว (฿3,000)
  - กระเป๋าสตางค์ หนังแท้ (฿600)
  - กระเป๋าเอกสาร สีดำ (฿1,200)
  - กระเป๋าสะพายยาว สีครีม (฿800)
  - กระเป๋าคลัทช์ สีทอง (฿700)
  - กระเป๋าเป้เด็ก ลายการ์ตูน (฿400)

### 4. เครื่องประดับ (11 รายการ)
- ราคา: ฿300 - ฿20,000
- สต็อก: 3-40 ชิ้น
- รายการ:
  - นาฬิกาข้อมือ (฿2,500)
  - สร้อยคอทอง (฿10,000) - สต็อกน้อย
  - แหวนเพชร (฿20,000) - สต็อกน้อย
  - ต่างหูเงิน (฿500)
  - กำไลทอง (฿15,000) - สต็อกน้อย
  - สร้อยข้อมือเงิน (฿700)
  - จี้พระ (฿5,000)
  - แหวนแฟชั่น (฿300)
  - นาฬิกาอัจฉริยะ (฿4,000)
  - เข็มกลัดเพชร (฿1,200)

### 5. อิเล็กทรอนิกส์ (11 รายการ)
- ราคา: ฿200 - ฿16,000
- สต็อก: 10-60 ชิ้น
- รายการ:
  - สมาร์ทโฟน (฿16,000)
  - หูฟังบลูทูธ (฿1,000)
  - แท็บเล็ต 10 นิ้ว (฿8,000)
  - ลำโพงบลูทูธ (฿1,500)
  - เมาส์ไร้สาย (฿400)
  - คีย์บอร์ดเกมมิ่ง (฿2,000)
  - เว็บแคม HD (฿1,200)
  - แบตเตอรี่สำรอง (฿700)
  - สายชาร์จเร็ว (฿200)
  - ที่ชาร์จไร้สาย (฿800)

## สถิติ

- **จำนวนสินค้าทั้งหมด:** 56 รายการ (รวมสินค้าเดิม)
- **จำนวนสินค้าใหม่:** 46 รายการ
- **ราคาต่ำสุด:** ฿200 (สายชาร์จเร็ว, รองเท้าแตะ)
- **ราคาสูงสุด:** ฿20,000 (แหวนเพชร)
- **สต็อกรวม:** ~1,500 ชิ้น

## ฟีเจอร์พิเศษ

### สินค้าราคาสูง (> ฿10,000)
- แหวนเพชร (฿20,000) - สต็อก 3 ชิ้น
- สมาร์ทโฟน (฿16,000) - สต็อก 10 ชิ้น
- กำไลทอง (฿15,000) - สต็อก 4 ชิ้น
- สร้อยคอทอง (฿10,000) - สต็อก 5 ชิ้น

### สินค้าสต็อกน้อย (≤ 5 ชิ้น)
- แหวนเพชร - 3 ชิ้น
- กำไลทอง - 4 ชิ้น
- สร้อยคอทอง - 5 ชิ้น

### สินค้าสต็อกมาก (≥ 50 ชิ้น)
- เสื้อยืดคอกลม สีขาว - 50 ชิ้น
- รองเท้าแตะ สีดำ - 60 ชิ้น
- เมาส์ไร้สาย - 50 ชิ้น
- สายชาร์จเร็ว - 60 ชิ้น

## การใช้งาน

### เพิ่มสินค้าใหม่
```bash
cd backend
node seed-products.js
```

### ตรวจสอบสินค้าในฐานข้อมูล
```bash
mysql -u root -p itkmmshop -e "
SELECT COUNT(*) as total FROM products;
"
```

### ดูสินค้าตามหมวดหมู่
```bash
# API
curl "http://localhost:5050/api/products?category_id=1"
```

## หมายเหตุ

- ราคาทั้งหมดคำนวณ VAT 7% แล้ว
- SKU เป็น unique key
- Script จะข้ามสินค้าที่มี SKU ซ้ำ
- สินค้าทั้งหมดมีสถานะ 'active'
- ไม่มีรูปภาพ (image_path = null)

## การทดสอบ

### ทดสอบการกรอง
```bash
# กรองตามหมวดหมู่
curl "http://localhost:5050/api/products?category_id=5"

# กรองตามราคา
curl "http://localhost:5050/api/products?sort_by=price_including_vat&sort_order=ASC"

# ค้นหา
curl "http://localhost:5050/api/products?search=เสื้อ"
```

### ทดสอบการเพิ่มในตระกร้า
```bash
curl -X POST http://localhost:5050/api/cart/add \
  -H "Content-Type: application/json" \
  -H "X-Session-ID: test_123" \
  -d '{"product_id":1,"quantity":2}'
```
